기능 소개

편리한 바로가기

'아스 - Ajou Swift'는 한번에 클릭으로 아주대학교의 다양한 웹페이지에 접속이 가능합니다.

12개의 바로가기

자주가는 학교 홈페이지의 바로가기를 클릭하여 접속하세요.

나의 학과

나의 학과를 등록하여 학과 홈페이지에 간편하게 접속하세요.

강력한 검색

'아스 - Ajou Swift'는 추가적인 학교 정보를 얻을 수 있는 강력한 교내 검색 기능을 제공합니다.

원클릭 검색

검색어를 입력하고 누르세요. 당신은 학교 어디로든 떠날 수 있습니다.

교직원 정보

교직원의 정보가 필요하다면 교직원의 이름을 검색해보세요.

매일매일 학사일정

'아스 - Ajou Swift'는 당신에게 필요한 학사일정을 매일 알려드립니다.

오늘의 학사일정

오늘의 일정이 궁금하다면 '아스 - Ajou Swift'를 클릭해보세요.

상세 학사일정

학사일정을 클릭하여 상세 학사일정을 확인해보세요.

사용자 설정

옵션 페이지를 통해 사용자는 자신에게 필요한 바로가기를 선택하여 나만의 '아스 - Ajou Swift'를 구성할 수 있습니다.

테마 선택

사용자는 옵션 페이지를 통해 자신이 원하는 테마를 선택할 수 있습니다. 라이트 테마와 다크 테마, 그리고 치토 테마(출시 예정)를 즐겨보세요!

안전한 보안

'아스 - Ajou Swift'는 서비스 구성을 위해 개인정보를 수집하지 않으며, 모든 사용자 설정은 크롬 스토리지에 안전하게 저장됩니다.

오픈소스

'아스 - Ajou Swift'는 항상 발전을 추구하고 있습니다. GitHub Repo를 통해 부족한 점을 알려주시고, 새로운 기능을 제안해주세요!
